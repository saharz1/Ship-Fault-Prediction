Ship Fault Prediction — Final Project
Overview
Machine learning project predicting ship faults using operational and environmental data, built as part of a B.Sc. in Industrial Engineering & Management (Big Data Specialization) at Ruppin Academic Center.
What's Inside

Data preprocessing and feature engineering
Statistical analysis and exploratory data analysis (EDA)
Machine learning models with cross-validation
Visualization of results and insights

Tools & Technologies
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Google Colab
Results
Successfully predicted ship faults with high accuracy, delivering actionable insights from real operational data.
